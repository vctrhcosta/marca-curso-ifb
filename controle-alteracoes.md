# Controle de Alterações — Memorial V4

- **[FEITO]** = já aplicado na V4

Quando for fechar o arquivo final para submissão, percorra cada item. Nenhum **[PENDENTE]** deve sobreviver.

## VI. Checklist pré-submissão

Percorrer antes de gerar o PDF/DOCX final:

- [ ] Todos os `[IMAGEM: ...]` substituídos por imagens reais
- [ ] Todos os `[A definir]` e `[X]` preenchidos ou removidos
- [ ] Todos os `<!-- NOTA: ... -->` removidos do arquivo final
- [ ] Nenhum item [PENDENTE] restante neste controle
- [ ] Todos os itens [VERIFICAR] confirmados
- [ ] PDF gerado e revisado visualmente (paginação, quebras, tabelas)
- [ ] Imagens em resolução adequada para impressão (300 dpi mínimo)
- [ ] Conferir se referências bibliográficas batem com as notas de rodapé
- [ ] Leitura final em voz alta para pegar tropeços de ritmo
